🚀 COMO USAR
═══════════════════════════════════════════════════════════════════════════

1️⃣  TESTAR LOCALMENTE (Imediato)
    $ python3 -m http.server 8000
    Abra: http://localhost:8000

2️⃣  CLONAR REPOSITÓRIO
    $ git clone https://github.com/seu-usuario/seu-repo.git
    $ cd seu-repo

3️⃣  CUSTOMIZAR
    • Editar nome da loja em index.html
    • Adicionar produtos em products.js
    • Trocar cores em style.css
    • Adicionar links de afiliados

4️⃣  PUBLICAR NO GITHUB PAGES
    $ git add .
    $ git commit -m "E-commerce Suplementos Premium"
    $ git push origin main
    ✨ Seu site estará em: https://seu-usuario.github.io/seu-repo
